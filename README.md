#  intern-project

> React Native app built using [`@react-native-community/cli`](https://github.com/react-native-community/cli)

---

##  How to Run

### 1. Start Metro

```sh
# Using npm
npm start

# OR using Yarn
yarn start
```

### 2. Run on Device/Emulator

#### Android

```sh
npm run android
# OR
yarn android
```

#### iOS (Mac only)

```sh
bundle install
bundle exec pod install

npm run ios
# OR
yarn ios
```
